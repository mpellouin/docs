---
title: "Register user"
api: "POST https://trotter.francecentral.cloudapp.azure.com/auth/register"
description: "This endpoint registers a new user to Trotter"
---

### Body

<ParamField body="username" type="string">
  This is the username desired by the registering user.
</ParamField>

<ParamField body="password" type="string">
  This is the password of the account that will be created.
</ParamField>

<ParamField body="email" type="string">
  This is the email of the account that will be created.
</ParamField>

### Response (on success)

<ResponseField name="accessToken" type="string">
  The accessToken to be used to access private routes of the API.
</ResponseField>


<RequestExample>

```bash Example Request
curl --location --request POST 'https://trotter.francecentral.cloudapp.azure.com/auth/register' \
--header 'Content-Type: application/json' \
--data-raw '{
    "username": "",
    "password": "",
    "email": ""
}'
```

</RequestExample>
