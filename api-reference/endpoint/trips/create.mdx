---
title: "Save Trip"
api: "POST https://trotter.francecentral.cloudapp.azure.com/trips"
description: "Saves a trip for a user"
---

### Params

<ParamField type="string" header="Authorization">
    Header in form of Bearer \<jwt\>
</ParamField>

### Body

<ParamField body="startDate" type="number" required>
The start date of the trip in number form.
</ParamField>

<ParamField body="endDate" type="number" required>
The end date of the trip in number form.
</ParamField>

<ParamField body="housingCoordinates" type="number[2]">
The coordinates of the traveler's housing.
</ParamField>

<ParamField body="cityName" type="string" required>
The name of the city the traver is going to.
</ParamField>

<ParamField body="tripData" type="string" required>
The data of the trip (composed of the POIs and the routes). This must be the JSON response from /IA call that is stringified. In JS, JSON.stringify(data). 
</ParamField>

<ParamField body="isShared" type="boolean" default="false">
If the trip is lookable for other users. Usually set to false and set to true once the suer decides to share it.
</ParamField>

### Response

empty response
