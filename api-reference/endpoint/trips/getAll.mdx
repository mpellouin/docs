---
title: "Get Trips"
api: "GET https://trottapi.onrender.com/trips"
description: "This endpoint gets all the trips of specified user"
---

### Header

<ParamField type="string" header="Authorization">
    Header in form of Bearer \<jwt\>
</ParamField>

### Response
**An array of:**
<ResponseField name="ownerId" type="string">
    The id of the response owner
</ResponseField>
<ResponseField name="startDate" type="number">
    The startDate in number form.
</ResponseField>
<ResponseField name="endDate" type="number">
    The endDate in number form.
</ResponseField>
<ResponseField name="housingCoordinates" type="number[2] | null">
    The coordinates of the hotel. Can be null
</ResponseField>
<ResponseField name="cityName" type="string">
</ResponseField>
<ResponseField name="tripData" type="string">
    The trip data in form of a JSON string. In JS to set back to JSON use JSON.parse(string).
</ResponseField>
<ResponseField name="isShared" type="boolean">
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://trottapi.onrender.com/trips' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <token>' \
```

</RequestExample>
